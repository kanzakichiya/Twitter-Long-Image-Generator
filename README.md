# 推特长图生成 (Twitter Long Image Generator)

一个用于将 Twitter (X) 推文转换为高质量长图片的油猴脚本。

### ✨ 主要功能

* **📸 一键生成**：在推文操作栏添加下载图标，点击即刻生成。
* **🧵 完整对话链**：在详情页可捕获最多 50 层上级对话，生成完整长图。
* **😃 统一 Emoji**：集成 **Twemoji**，确保 Emoji 在各平台显示一致。
* **💎 高清渲染**：使用 **SnapDOM** 技术，生成的图片更清晰锐利。

### 🚀 安装指南

1.  安装 [Tampermonkey](https://www.tampermonkey.net/) 浏览器扩展。
2.  点击[这里](https://github.com/kanzakichiya/Twitter-Long-Image-Generator/raw/refs/heads/main/%E6%8E%A8%E7%89%B9%E9%95%BF%E5%9B%BE%E7%94%9F%E6%88%90-1.0.user.js)安装脚本。
3.  访问 [Twitter/X](https://twitter.com) 并刷新页面。

### 📖 使用方法

1.  找到任意推文。
2.  点击操作栏中的 **下载图标**。
3.  稍等片刻，PNG图片将自动下载。
